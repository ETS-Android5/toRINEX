Detailed Description

SettingsActivity, the main activity for this app, is a PreferenceActivity that presents the menu with the list of application settings and activities that can be performed. On handset devices, settings are presented as a single list. On tablets,settings are split by category, with category headers shown to the left of the list of settings.

See Android Design: Settings for design guidelines and the Settings API Guide for more information on developing a Settings UI.
